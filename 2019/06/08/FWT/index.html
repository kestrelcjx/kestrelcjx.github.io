快速沃尔什变换(FWT)

  • 数论

快速沃尔什变换(FWT)

用途：求
$$C(i) = \sum_{i=j@k}A(j)*B(k)$$
其中@为位运算或(|), 与(&), 异或(^)中一种,算法复杂度O($n\log(n)$)。

1. 或运算

令 $$FWT[A] = (A_1, A_2…)$$
其中
$$A_i = \sum_{i=i|j}A[j]$$
用A0表示A的前一半，A1表示A的后一半，那么如果已经求得FWT[A0]和FWT[A1]，则可以得到
FWT[A] = (FWT[A0], FWT[A0]+FWT[A1]) (‘+’ 对应位相加) 当A长度为1时，FWT[A] = A(A0最高
位为0，对FWT[A]后一半能贡献，而A1最高位为1，对FWT[A]前一半不能贡献)。所以，就可以
通过迭代分治，在O(nlogn)求得FWT[A]。
并且，由
$$C(i) = \sum_{i=j|k}A(j)*B(k)$$
得到FWT[C] = FWT[A]*FWT[B] (‘*’ 对应位相乘)，所以能在求得FWT[A]和FWT[B]基础上，在
O(n)求得FWT[C]。
最后，需要将FWT[C]恢复为C。通过观察求FWT[A]的过程，发现只需要将FWT[C]的后半部分减
去FWT[C]的前半部分IFWT[C] = (IFWT[C0], IFWT[C1]-IFWT[C0]) 当C长度为1时，IFWT[C] = C
就可以得到C。所以，同样能通过逆向迭代分治，在O(nlogn)求得C。

2. 与运算

FWT的与运算和或运算非常类似，只需改变FWT[A]中元素含义，让FWT[A]中
$$A_i = \sum_{i=i\&j}A[j]$$
同样我们能发现FWT[A] = (FWT[A0]+FWT[A1], FWT[A1]),同时FWT[C] = FWT[A]*FWT[B]
所以同样能求得FWT[C],最后由IFWT[C] = (IFWT[C0]-IFWT[C1], IFWT[C1]),求得C

3. 异或运算

FWT的异或运算相比或运算和与运算有些不同，我们设i&j所得结果中1的个数的奇偶表示i&j的
奇偶性(偶：0；奇：1)，那么会有(i&k的奇偶性)^(j&k的奇偶性) = (i^j)&k的奇偶性，那么，令
$$A_i = \sum_{i\&k=0}A[k]-\sum_{i\&k=1}A[k]$$
其中i&k=0表示i&k为偶性，i&k=1表示i&k为奇性，我们能发现仍然有FWT[C] = FWT[A]*FWT[B]
假设不考虑二进制最高位已经求得FWT[A0]和FWT[A1],要求FWT[A]。那么再多考虑最高位，A0中
最高位为0，FWT[A0]仍然能直接贡献到FWT[A0]和FWT[A1]；而A1最高位为1，FWT[A1]能直
接贡献到FWT[A0]，但贡献到FWT[A1]时，因为有了最高位1，i&k的奇偶性改变。所以，有
FWT[A] = (FWT[A0]+FWT[A1], FWT[A0]-FWT[A1])最后求得FWT[C]后，由
IFWT[C] = ((IFWT[C0]+IFWT[C1])/2, (IFWT[C0]-IFWT[C1])/2),恢复得到C

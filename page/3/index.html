е®һзҺ°

йҰ–е…ҲпјҢе°ҶжҹҗдёҖйқһжҢҮе®ҡзӮ№дҪңдёәиҷҡж ‘зҡ„ж №пјҢе°Ҷе…¶ж”ҫе…Ҙж Ҳдёӯпјӣз„¶еҗҺпјҢжұӮеҮәж ‘дёҠзӮ№зҡ„dfsеәҸпјҢе°ҶжҢҮе®ҡзҡ„зӮ№жҢүdfsеәҸд»Һе°ҸеҲ°еӨ§жҺ’еәҸеҗҺпјҢдҫқж¬Ўж”ҫе…Ҙж ҲдёӯгҖӮеңЁе°ҶзӮ№ж”ҫе…Ҙж Ҳж—¶пјҢеҲҶжғ…еҶөи®Ёи®әпјҡ
1.еҰӮжһңж Ҳдёӯе…ғзҙ дёӘж•°е°ҸдәҺзӯүдәҺ1дёӘпјҢзӣҙжҺҘе…Ҙж ҲпјҢиҝ”еӣһпјӣеҗҰеҲҷжұӮеҮәж ҲйЎ¶е…ғзҙ дёҺе…Ҙж Ҳе…ғзҙ зҡ„LCAгҖӮ
2.еҰӮжһңж ҲйЎ¶е…ғзҙ дёәLCAпјҢзӣҙжҺҘиҝ”еӣһпјҢе…ғзҙ дёҚйңҖиҰҒе…Ҙж ҲпјӣеҰӮжһңж ҲйЎ¶е…ғзҙ дёҚдёәLCAпјҢеҫӘзҺҜеҲӨж–ӯж Ҳдёӯе…ғзҙ дёӘж•°еӨ§дәҺ1дё”з¬¬дәҢдёӘе…ғзҙ dfsеәҸдёҚе°ҸдәҺLCAзҡ„dfsеәҸпјҢе°Ҷж ҲйЎ¶е…ғзҙ е’Ңж Ҳдёӯз¬¬дәҢдёӘе…ғзҙ е»әиҫ№е№¶е°Ҷж ҲйЎ¶е…ғзҙ еҮәж ҲпјҢеҗҰеҲҷи·іеҮәеҫӘзҺҜпјӣ
3.еҰӮжһңеңЁеҫӘзҺҜз»“жқҹпјҢж ҲйЎ¶е…ғзҙ дёҚдёәLCAпјҢе°Ҷж ҲйЎ¶е…ғзҙ дёҺLCAе»әиҫ№пјҢж ҲйЎ¶е…ғзҙ еҮәж ҲпјҢеҶҚе°ҶLCAе…Ҙж Ҳпјӣ

жұӮи§ЈжңҖе°Ҹж–ҜеқҰзәіж ‘дҪҝз”ЁеҠЁжҖҒи§„еҲ’жҖқжғіпјҢз”Ёdp[S][u]иЎЁзӨәд»Ҙuдёәж №иҠӮзӮ№гҖҒзҠ¶жҖҒSзҡ„жңҖе°ҸиҠұиҙ№пјҢSзҡ„дәҢиҝӣеҲ¶еҗ„дҪҚиЎЁзӨәжҳҜеҗҰеҢ…еҗ«еҗ„зӮ№гҖӮ

1.е°ҶSеҲҶдёәдёӨйғЁеҲҶ:dp[S][u] = min(dp[S][u], dp[S0][u]+dp[S^S0][u])йҖҡиҝҮжһҡдёҫеҗ„з§ҚиҪ¬з§»зҠ¶жҖҒеҚіеҸҜгҖӮ
2.йҖҡиҝҮиҫ№u-vиҪ¬з§»:dp[S][u] = min(dp[S][u], dp[S][v]+w[u][v])йҖҡиҝҮжңҖзҹӯи·Ҝз®—жі•иҝӣиЎҢиҪ¬з§»гҖӮ

2019-06-08

еҝ«йҖҹжІғе°”д»ҖеҸҳжҚў(FWT)

з”ЁйҖ”пјҡжұӮ
$$C(i) = \sum_{i=j@k}A(j)*B(k)$$
е…¶дёӯ@дёәдҪҚиҝҗз®—жҲ–(|), дёҺ(&), ејӮжҲ–(^)дёӯдёҖз§Қ,з®—жі•еӨҚжқӮеәҰO($n\log(n)$)гҖӮ

1. жҲ–иҝҗз®—

д»Ө $$FWT[A] = (A_1, A_2вҖҰ)$$
е…¶дёӯ
$$A_i = \sum_{i=i|j}A[j]$$
з”ЁA0иЎЁзӨәAзҡ„еүҚдёҖеҚҠпјҢA1иЎЁзӨәAзҡ„еҗҺдёҖеҚҠпјҢйӮЈд№ҲеҰӮжһңе·Із»ҸжұӮеҫ—FWT[A0]е’ҢFWT[A1]пјҢеҲҷеҸҜд»Ҙеҫ—еҲ°
FWT[A] = (FWT[A0], FWT[A0]+FWT[A1]) (вҖҳ+вҖҷ еҜ№еә”дҪҚзӣёеҠ ) еҪ“Aй•ҝеәҰдёә1ж—¶пјҢFWT[A] = A(A0жңҖй«ҳ
дҪҚдёә0пјҢеҜ№FWT[A]еҗҺдёҖеҚҠиғҪиҙЎзҢ®пјҢиҖҢA1жңҖй«ҳдҪҚдёә1пјҢеҜ№FWT[A]еүҚдёҖеҚҠдёҚиғҪиҙЎзҢ®)гҖӮжүҖд»ҘпјҢе°ұеҸҜд»Ҙ
йҖҡиҝҮиҝӯд»ЈеҲҶжІ»пјҢеңЁO(nlogn)жұӮеҫ—FWT[A]гҖӮ
е№¶дё”пјҢз”ұ
$$C(i) = \sum_{i=j|k}A(j)*B(k)$$
еҫ—еҲ°FWT[C] = FWT[A]*FWT[B] (вҖҳ*вҖҷ еҜ№еә”дҪҚзӣёд№ҳ)пјҢжүҖд»ҘиғҪеңЁжұӮеҫ—FWT[A]е’ҢFWT[B]еҹәзЎҖдёҠпјҢеңЁ
O(n)жұӮеҫ—FWT[C]гҖӮ
жңҖеҗҺпјҢйңҖиҰҒе°ҶFWT[C]жҒўеӨҚдёәCгҖӮйҖҡиҝҮи§ӮеҜҹжұӮFWT[A]зҡ„иҝҮзЁӢпјҢеҸ‘зҺ°еҸӘйңҖиҰҒе°ҶFWT[C]зҡ„еҗҺеҚҠйғЁеҲҶеҮҸ
еҺ»FWT[C]зҡ„еүҚеҚҠйғЁеҲҶIFWT[C] = (IFWT[C0], IFWT[C1]-IFWT[C0]) еҪ“Cй•ҝеәҰдёә1ж—¶пјҢIFWT[C] = C
е°ұеҸҜд»Ҙеҫ—еҲ°CгҖӮжүҖд»ҘпјҢеҗҢж ·иғҪйҖҡиҝҮйҖҶеҗ‘иҝӯд»ЈеҲҶжІ»пјҢеңЁO(nlogn)жұӮеҫ—CгҖӮ

2. дёҺиҝҗз®—

FWTзҡ„дёҺиҝҗз®—е’ҢжҲ–иҝҗз®—йқһеёёзұ»дјјпјҢеҸӘйңҖж”№еҸҳFWT[A]дёӯе…ғзҙ еҗ«д№үпјҢи®©FWT[A]дёӯ
$$A_i = \sum_{i=i\&j}A[j]$$
еҗҢж ·жҲ‘д»¬иғҪеҸ‘зҺ°FWT[A] = (FWT[A0]+FWT[A1], FWT[A1]),еҗҢж—¶FWT[C] = FWT[A]*FWT[B]
жүҖд»ҘеҗҢж ·иғҪжұӮеҫ—FWT[C],жңҖеҗҺз”ұIFWT[C] = (IFWT[C0]-IFWT[C1], IFWT[C1]),жұӮеҫ—C

3. ејӮжҲ–иҝҗз®—

FWTзҡ„ејӮжҲ–иҝҗз®—зӣёжҜ”жҲ–иҝҗз®—е’ҢдёҺиҝҗз®—жңүдәӣдёҚеҗҢпјҢжҲ‘д»¬и®ҫi&jжүҖеҫ—з»“жһңдёӯ1зҡ„дёӘж•°зҡ„еҘҮеҒ¶иЎЁзӨәi&jзҡ„
еҘҮеҒ¶жҖ§(еҒ¶пјҡ0пјӣеҘҮпјҡ1)пјҢйӮЈд№Ҳдјҡжңү(i&kзҡ„еҘҮеҒ¶жҖ§)^(j&kзҡ„еҘҮеҒ¶жҖ§) = (i^j)&kзҡ„еҘҮеҒ¶жҖ§пјҢйӮЈд№ҲпјҢд»Ө
$$A_i = \sum_{i\&k=0}A[k]-\sum_{i\&k=1}A[k]$$
е…¶дёӯi&k=0иЎЁзӨәi&kдёәеҒ¶жҖ§пјҢi&k=1иЎЁзӨәi&kдёәеҘҮжҖ§пјҢжҲ‘д»¬иғҪеҸ‘зҺ°д»Қз„¶жңүFWT[C] = FWT[A]*FWT[B]
еҒҮи®ҫдёҚиҖғиҷ‘дәҢиҝӣеҲ¶жңҖй«ҳдҪҚе·Із»ҸжұӮеҫ—FWT[A0]е’ҢFWT[A1],иҰҒжұӮFWT[A]гҖӮйӮЈд№ҲеҶҚеӨҡиҖғиҷ‘жңҖй«ҳдҪҚпјҢA0дёӯ
жңҖй«ҳдҪҚдёә0пјҢFWT[A0]д»Қз„¶иғҪзӣҙжҺҘиҙЎзҢ®еҲ°FWT[A0]е’ҢFWT[A1]пјӣиҖҢA1жңҖй«ҳдҪҚдёә1пјҢFWT[A1]иғҪзӣҙ
жҺҘиҙЎзҢ®еҲ°FWT[A0]пјҢдҪҶиҙЎзҢ®еҲ°FWT[A1]ж—¶пјҢеӣ дёәжңүдәҶжңҖй«ҳдҪҚ1пјҢi&kзҡ„еҘҮеҒ¶жҖ§ж”№еҸҳгҖӮжүҖд»ҘпјҢжңү
FWT[A] = (FWT[A0]+FWT[A1], FWT[A0]-FWT[A1])жңҖеҗҺжұӮеҫ—FWT[C]еҗҺпјҢз”ұ
IFWT[C] = ((IFWT[C0]+IFWT[C1])/2, (IFWT[C0]-IFWT[C1])/2),жҒўеӨҚеҫ—еҲ°C

  3. protected void finalize() 当垃圾收集确定不再有该对象的引用时，垃圾收集器在对象上调用该类

  9. void wait() 导致当前线程等待，直到另一个线程调用该对象的notify()或notifyAll()方法
  10. void wait(long timeout) 导致当前线程等待，直到另一个线程调用 notify()方法或该对象的 notifyAll()方法，或者指定的时间已过。
  11. void wait(long timeout, int nanos)导致当前线程等待，直到另一个线程调用该对象的 notify()方法或 notifyAll()方法，或者某些其他线程中断当前线程，或一定量的实时时间。

虚树

2019-08-27

虚树

用途：在树上做DP时，有时候我们并不需要从某些点进行转移，只需要从某些关键点便可求得答案，多余的点会增加算法的时间复杂度。虚树就是在原来的树基础上去掉多余点、只保留关键点，关键点包括指定的点以及他们的最近公共祖先(LCA)。

实现

首先，将某一非指定点作为虚树的根，将其放入栈中；然后，求出树上点的dfs序，将指定的点按dfs序从小到大排序后，依次放入栈中。在将点放入栈时，分情况讨论：
1.如果栈中元素个数小于等于1个，直接入栈，返回；否则求出栈顶元素与入栈元素的LCA。
2.如果栈顶元素为LCA，直接返回，元素不需要入栈；如果栈顶元素不为LCA，循环判断栈中元素个数大于1且第二个元素dfs序不小于LCA的dfs序，将栈顶元素和栈中第二个元素建边并将栈顶元素出栈，否则跳出循环；
3.如果在循环结束，栈顶元素不为LCA，将栈顶元素与LCA建边，栈顶元素出栈，再将LCA入栈；
4.将指定点入栈。

斯坦纳树(Steiner Tree)

2019-08-26

斯坦纳树(Steiner Tree)

用途：在图中求包含特定点的生成树，可以包含额外的点，最小生成树属于最小斯坦纳的一种。
算法复杂度O($n*3^k$)，n为图顶点个数，k为需要包含的特殊点个数。

求解最小斯坦纳树

求解最小斯坦纳树使用动态规划思想，用dp[S][u]表示以u为根节点、状态S的最小花费，S的二进制各位表示是否包含各点。
转移方式有两种：
1.将S分为两部分:dp[S][u] = min(dp[S][u], dp[S0][u]+dp[S^S0][u])通过枚举各种转移状态即可。
2.通过边u-v转移:dp[S][u] = min(dp[S][u], dp[S][v]+w[u][v])通过最短路算法进行转移。

快速沃尔什变换(FWT)

2019-06-08

快速沃尔什变换(FWT)

用途：求
$$C(i) = \sum_{i=j@k}A(j)*B(k)$$
其中@为位运算或(|), 与(&), 异或(^)中一种,算法复杂度O($n\log(n)$)。

1. 或运算

令 $$FWT[A] = (A_1, A_2…)$$
其中
$$A_i = \sum_{i=i|j}A[j]$$
用A0表示A的前一半，A1表示A的后一半，那么如果已经求得FWT[A0]和FWT[A1]，则可以得到
FWT[A] = (FWT[A0], FWT[A0]+FWT[A1]) (‘+’ 对应位相加) 当A长度为1时，FWT[A] = A(A0最高
位为0，对FWT[A]后一半能贡献，而A1最高位为1，对FWT[A]前一半不能贡献)。所以，就可以
通过迭代分治，在O(nlogn)求得FWT[A]。
并且，由
$$C(i) = \sum_{i=j|k}A(j)*B(k)$$
得到FWT[C] = FWT[A]*FWT[B] (‘*’ 对应位相乘)，所以能在求得FWT[A]和FWT[B]基础上，在
O(n)求得FWT[C]。
最后，需要将FWT[C]恢复为C。通过观察求FWT[A]的过程，发现只需要将FWT[C]的后半部分减
去FWT[C]的前半部分IFWT[C] = (IFWT[C0], IFWT[C1]-IFWT[C0]) 当C长度为1时，IFWT[C] = C
就可以得到C。所以，同样能通过逆向迭代分治，在O(nlogn)求得C。

2. 与运算

FWT的与运算和或运算非常类似，只需改变FWT[A]中元素含义，让FWT[A]中
$$A_i = \sum_{i=i\&j}A[j]$$
同样我们能发现FWT[A] = (FWT[A0]+FWT[A1], FWT[A1]),同时FWT[C] = FWT[A]*FWT[B]
所以同样能求得FWT[C],最后由IFWT[C] = (IFWT[C0]-IFWT[C1], IFWT[C1]),求得C

3. 异或运算

FWT的异或运算相比或运算和与运算有些不同，我们设i&j所得结果中1的个数的奇偶表示i&j的
奇偶性(偶：0；奇：1)，那么会有(i&k的奇偶性)^(j&k的奇偶性) = (i^j)&k的奇偶性，那么，令
$$A_i = \sum_{i\&k=0}A[k]-\sum_{i\&k=1}A[k]$$
其中i&k=0表示i&k为偶性，i&k=1表示i&k为奇性，我们能发现仍然有FWT[C] = FWT[A]*FWT[B]
假设不考虑二进制最高位已经求得FWT[A0]和FWT[A1],要求FWT[A]。那么再多考虑最高位，A0中
最高位为0，FWT[A0]仍然能直接贡献到FWT[A0]和FWT[A1]；而A1最高位为1，FWT[A1]能直
接贡献到FWT[A0]，但贡献到FWT[A1]时，因为有了最高位1，i&k的奇偶性改变。所以，有
FWT[A] = (FWT[A0]+FWT[A1], FWT[A0]-FWT[A1])最后求得FWT[C]后，由
IFWT[C] = ((IFWT[C0]+IFWT[C1])/2, (IFWT[C0]-IFWT[C1])/2),恢复得到C
